Page Details:

This class offers some utilities to work with existing Word (.docx) documents Among another useful options, it allows for the merging of two docx with certain configuration options, the replacement and highlighting of text as well as removing selected contents, adding and removing watermarks, replace chart variables and add WordML content at the end of an existing Word file.
